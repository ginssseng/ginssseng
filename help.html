Конструктор
"Домик Кроша"

С помощью элементов из конструктора бренда «Город Мастеров» вы соберете для него уютное жилище. На дверке домика будет изображена морковка, так как это любимое лакомство Кроша. Все детали тщательно обработаны, поэтому ими невозможно порезаться. В комплекте есть фигурка Кроша, а значит, как только домик будет готов, можно начать увлекательную сюжетную игру.

Цена: 800 руб

Конструктор
"Модная Нюша"

Из деталей конструктора девочка соберет уютный домик для обаятельной героини мультика «Смешарики». Детали выполнены из высококачественного пластика и тщательно обработаны, поэтому легко крепятся друг к другу и хорошо держатся. Как только домик будет готов, ребенок сможет начать увлекательную сюжетную игру, ведь в комплекте есть фигурка его будущей обаятельной хозяйки.
